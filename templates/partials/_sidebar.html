{# partials/_sidebar.html ---------------------- The left navigation sidebar, shared by every dashboard-area page. WHY A SEPARATE PARTIAL FILE? ------------------------------- Every future page (Add Trainee, View Trainees, Reports...) will reuse this exact same sidebar. If we copy-pasted this markup into every page template, changing one menu item would mean editing it in ten places. By keeping it in one file and {% include %}-ing it, we edit it ONCE and every page picks up the change automatically - the DRY principle ("Don't Repeat Yourself"). HOW "ACTIVE PAGE" HIGHLIGHTING WORKS ---------------------------------------- Each route that renders a page using this sidebar passes an `active_page` variable (e.g. active_page="dashboard" from routes/dashboard.py). Below, each link compares its own name against that variable with a simple Jinja `if` - if they match, we add Bootstrap's "active" class, which our CSS styles with the blue highlight. #}
